# Як перевірити: List через enumerator, Sort default, Sort альтернативне, Stats

> *Перед безпосередною перевіркою цієї функціональності необхідно виконати підготовку:*
> 1. Додати доставки до менеджера доставок (1 пункт меню).
> 2. Змінити статус деяких доставок на скасовано чи доставлено (5 і 6 пункти меню).
> 3. Завантажити доставки з менеджера до колекції (10 пункт меню).
> 4. Додати декілька доставок на різні позиції (12 пункт меню).

- Для перевірки роботи **перебору доставок за допомогою enumerator** потрібно обрати 
15 пункт меню і переглянути, чи присутні там всі додані раніше доставки.

- Для перевірки **базового сортування** потрібно обрати 17 пункт меню і переглянути, 
чи всі доставки розташовані у порядку зростання їх ID.

- Для перевірки **альтернативного сортування** потрібно обрати 18 пункт меню і переглянути, 
чи всі доставки розташовані у порядку спадання їх пріоритету та зростання їх ID (в межах 
одного пріоритету).

- Для перевірки **статистики** потрібно обрати 19 пункт меню і переглянути, 
чи співпадає кількість доставок загалом та за кожним статусом окремо з кількістю
доданих раніше доставок з відповідними характеристиками.
